Dans un premier temps j'ai mis en place ma bdd

J'ai crée ma bdd 'wcs' où j'ai crée une 'table' argonaute avec un id en auto-incrémentation ainsi que le nom.
J'ai rentré dans ma base de données les 3 nom des argonautes déjà connu.

J'ai ensuite crée le dossier wcs qui va contenir mon projet, dans mon dossier www de wamp64 pour travailler sur mon localhost
J'ai crée un dossier assets dans lequel je vais mettre un dossier img et css pour avoir un projet structuré.

Je crée un fichier connect.php pour pouvoir connecter ma bdd à mes futurs fichiers php.
Je crée un fichier header ainsi que footer pour pouvoir les appelés dans un fichier index.php, je n'ai pas voulu intégrer intégrer directement le header et le footer dans mon index.php pour une question de bonne pratique et dans l'optique d'un amélioration futur du site, si je viens à créer d'autre page je n'aurai juste qu'à les rappeler.
